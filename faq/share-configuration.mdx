Вы можете поделиться сервером из подписки (если она не зашифрована и не скрыта) или из локального списка серверов. Для этого выполните следующие шаги:

1. Проведите пальцем вправо по заголовку нужного сервера.
2. После свайпа отобразятся две иконки. Нажмите на фиолетовую иконку со стрелкой для выбора способа обмена.
3. В появившемся меню доступны три варианта:
   - **В буфер обмена**
   - **QR-код**
   - **JSON**

Выберите нужный вариант для обмена конфигурацией.
