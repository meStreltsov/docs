---
title: "加密链接"
---

应用支持加密链接。链接可以使用 **RSA-4096** 加密。现在推荐只使用 RSA-4096，它被称为 `happ://crypt3/`。 加密链接的目的是隐藏用户的订阅地址。添加加密订阅后，用户无法编辑、查看或分享订阅中包含的服务器配置。加密密钥本身被安全地内置在应用中，从而确保订阅数据的保护。。

您可以通过以下三种方式加密链接：

- 使用本[网页](https://crypto.happ.su/)
- 使用 [API](https://www.happ.su/main/ru/documentation/crypto-link#instrukciya-po-api)
- 使用 [RSA](https://www.happ.su/main/zh/dev-docs/crypto-link#rsa-mi-yao) 密钥

#### **API 使用说明**

要使用 API，您需要发送请求至以下地址： https://crypto.happ.su/api.php

示例：

```
curl -X POST -H "Content-Type: application/json" -d '{"url":"https://your_url.com"}' "https://crypto.happ.su/api.php"
```

结果将返回您链接的加密版本，可以在应用中使用。
